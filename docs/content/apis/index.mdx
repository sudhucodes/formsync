# FormSync Public API

FormSync provides a **Public API** that lets you fetch form submissions securely using an API key.

This API is designed for:
- Custom dashboards
- Backend services
- Analytics pipelines
- Admin panels

---

## Base URL

[https://api.formsync.app/public/v1](https://api.formsync.app/public/v1)

---

## Authentication

All requests must include your **API key** in the `Authorization` header.

```http
Authorization: Bearer fs_your_api_key_here
```

### Important Notes

* API keys are shown **only once**
* Store them securely
* Regenerate if lost or leaked

---

## Available APIs

Currently, FormSync exposes the following API:

* **Get Form Submissions**

More APIs (webhooks, exports, filters) will be added in the future.

> Continue to [Submissions API](/docs/apis/submissions) to get started.
