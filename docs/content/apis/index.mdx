# FormSync Public API

FormSync provides a **Public API** that lets you fetch form submissions securely using an API key.

This API is designed for:
- Custom dashboards
- Backend services
- Analytics pipelines
- Admin panels

---

## Base URL

[https://api.formsync.app/public/v1](https://api.formsync.app/public/v1)

---

## Available APIs

Currently, FormSync give the following API:

* **Get Form Submissions**

More APIs (webhooks, forms, filters) will be added in the future.

> Continue to [Submissions API](/docs/apis/submissions) to get started.
